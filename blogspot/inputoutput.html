Bagian 2 : Input & Output Dalam Bahasa C

Output Dalam Bahasa C

Untuk menampilkan sebuah tulisan berupa angka nomor ataupun simbol dalam console aplication. Kita perlu menambahkan sebuah code . Dalam bahasa C printf dan puts lah yang digunakan unutk menampilkan tulisan. Lantas apa bedanya printf dengan puts? Jika ingin tahu pahami penjelasan berikut.

Printf

Printf berfungsi untuk menampilkan segala sesuatu ke layar pengguna. Untuk menampilkan di layar ,tulisannya harus diketik didalam tanda " " dalam code printf("") dan jangan lupa diakhiri dengan ;

Output

Printf juga bisa menampilkan isi dari sebuah variabel. Fungsi penulisan printf untuk menampilkan variabel perlu disertakan format dari tipe data variabel itu .
  • %s --- String
  • %c --- Character
  • %d --- Integer
  • %f --- Float

Output

Dalam printf juga terdapat beberapa fungsi untuk mengatur tulisan yang ada didalamnya seperti
  • \t untuk membuat jarak satu tab
  • \n untuk membuat baris selanjutnya
  • %2.. untuk membuat jarak 2 digit, tanda .. dapat diisi dengan tipe data dari variabel yang nilainya akan ditampilkan

Puts

Puts memiliki fungsi yang sama dengan printf . yaitu menaapilkan tulisan pada layar. Namun bedanya penulisan puts tidak usah disertai dengan \n karena puts secara default menampilkan tulisan dengan jangkauan satu baris. Jadi apabila kita menggunakan 2 buah puts maka tulisannya akan otomatis berada dalam baris yang berbeda.

Output

Output Dalam Bahasa C

Terdapat 3 metode input dalam bahasa c yaitu scanf , gets , dan getchar.

Scanf

Scanf berfungsi menerima inputan data dari user melalui keyboard yang akan disimpan di variabel.
format penulisan :
scanf("%d",&nama_variabel);

Output

Gets

Gets digunakan untuk menerima inputan berupa kata atau kalimat. Keunggulan gets ketimbang scanf dalam penggunaan inputan string adalah gets dapat menyimpan 2 kata atau lebih yang memiliki spasi sedangkan scanf tidak dapat.
Format penulisan :
gets(nama_variabel);

Output

Getchar

Getchar berfungsi untuk menerima inputan berupa karakter.
Format penulisan :
nama_variabel = getchar();
