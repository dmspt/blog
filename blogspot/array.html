Bagian 7 : Array Dalam Bahasa C

Definisi Array

Array merupakan kumpulan dari nilai - nilai data bertipe sama dalam urutan tertentu dengan menggunakan nama yang sama. Array dapat diumpamakan seperti rak sepatu yang berisi berbagai sepatu. Rak tidak diperbolehkan diisi selain sepatu. Untuk memudahkan penandaan, maka terdapat alamat atau nomor yang merupakan alamat daru rak tersebut. Penomoran ini disebut index yang dimulai dari 0 bukan 1.

Array Satu Dimensi

Suatu array 1 dimensi dapat digambarkan sebagai kotak panjang yang berisi kotak - kotak kecil di dalam kotak panjang.

Deklarasi Array

Sebelum digunakan variabel array perlu dideklarasikan terlebih dahulu. Cara mendeklarasikan variabel array sama dengan deklarasi variabel lainnya, hanya saja diikuti dengan indeks yang menunjukan jumlah maksimum data yang disediakan.
Bentuk umum dari pendeklarasian array :
tipe_data nama_array[jumlah_elemen]
  • tipe_data : tipe data yang nanti akan disimpan dalam array ini
  • nama_array : nama dari array
  • jumlah_elemen : banyaknya data yang akan disimpan dalam array

Inisialisasi Array

Inisialisasi dilakukan dengan menyebut elemen - elemennya dengan dipisahkan oleh tanda koma dalam tanda kurung kurawal. Khusus untuk string, maka setiap elemen atau setiap karakter diberi tanda kutip tunggal. Contoh adalah sebagai berikut :
  • int bil[5] = { 1 , 2 , 3 , 4 , 5 };
  • char vokal[5] = {"a" , "i" , "u" , "e" , "o"};
Banyaknya elemen array dapat pula tidak dicantumkan pada saat inisialisasi. Kompilator nantinya akan menghitung sendiri banyaknya elemen array tersebut. Misalnya :
  • int bil[ ] = {2 , 6 , 10 , 9 , 8 };
  • char nama[ ] = {"u" , "d" , "i" , "n" , "u" , "s"};

Array Multidimensi

Sebuah array multidimensi adalah array yang berisi satu atau lebih array. Bentuk paling sederhanan dari array multidimensi adalah array dua dimensi. Array dua dimensi tersusun dalam bentuk baris dan kolom seperti sebuah matriks atau tabel.Indeks pertama menunjukan baris dan indeks kedua menunjukan kolom.

Deklarasi Array Multidimensi

Dalam bahasa C pendeklarasian variabel diawali dengan penulisan tipe data variabel, kemudian nama variabel tersebut, dan diakhiri dengan titik koma (;).
tipe_data nama_array[index-1][index-2]
